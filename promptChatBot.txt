Demander en français à ChatGPT de coder un site web

1 PRÉREQUIS

J'aimerais que vous agissiez en tant que développeur web frontal. Pour ce projet, vous coderez un nouveau site web en utilisant HTML, CSS et un peu de PHP. Le site doit être adapté aux mobiles et responsive. Il s'agira d'un site web pour un magasin vendant du matériel de roller hockey. Le site web aura 6 pages avec la même mise en page : une page d’accueil, une page de catalogue montrant les produits, une page montrant les détails d'un produit, une page de contact, une page de mentions légales et un panier d'achat. Toutes les pages partageront la même mise en page : un en-tête, un menu de navigation et un pied de page. L'en-tête contiendra un logo flottant à gauche, un titre centré et un lien vers le panier d'achat à droite. Ensuite, il y aura un menu de navigation central avec un lien vers la page d'accueil, la page des produits et la page de contact. Le pied de page présentera au centre un copyright avec le nom de la boutique et un petit menu à droite avec un lien vers la page de contact et la page légale. Toutes les pages css seront stockées dans un sous-répertoire appelé « css ». Toutes les pages web auront l'extension « php » au lieu de « html ». Toutes les images seront stockées dans un sous-répertoire appelé « img ». Ne fournissez aucune explication pour le code que vous écrivez. Avez-vous tout compris ? Veuillez répondre par oui ou par non si vous comprenez toutes les exigences.

2 Page d'index

Ma première demande est « Créer la page d'index du site web ». Cette page accueille le client sur le site web. Elle comporte un lien vers une page styles.css qui sera partagée pour l'ensemble du site et un autre lien vers une page css spécifique pour l'affichage de cette page. Vous devez inclure deux sections de contenu sur cette page, chacune avec une balise H2 pour commencer la section, ainsi que plusieurs balises de paragraphe après cela avec un texte pertinent sur le matériel de roller hockey. Vous devez également inclure une vidéo d'un match de roller hockey, accompagnée d'une légende expliquant le contenu de la vidéo.

8 Panier d'achat

Ma prochaine demande est « Créez la page du panier d'achat ». Ajoutez du code PHP qui récupérera les paramètres de la requête CGI (envoyés avec une requête GET ou POST) d’ajout d’un produit au panier, les mettra dans un tableau et stockera le tableau dans une variable de session.
Ensuite, dans la partie centrale de la page, le code affichera la variable de session sous la forme d'une liste de produits avec un peu de code HTML. 
Pour chaque produit, une image, son nom et son prix seront affichés. Ajoutez un bouton et du code PHP pour supprimer le produit.
Une variable calculera la somme totale des produits.
À la fin de la partie principale, vous affichez le montant total du panier et un bouton pour payer la commande.
